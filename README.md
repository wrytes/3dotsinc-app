# 3Dots Inc. Landing Page

Modern landing page for 3Dots Inc. - a privately held company focused on financial strategy, digital assets, and advanced market models.

## Tech Stack

- Next.js 15 (Pages Router)
- React 19
- TypeScript
- Tailwind CSS
- Avenir font

## Development

```bash
# Install dependencies
yarn install

# Run dev server
yarn dev
```

Visit [http://localhost:3000](http://localhost:3000)

## Build

```bash
yarn build
yarn start
```

## Components

- `Hero` - Main hero section with company tagline
- `TextSection` - Reusable centered text sections
- `ImageTextSection` - Image + text layouts
- `Protocols` - Partner protocol showcase
- `FAQs` - Frequently asked questions
- `Partnership` - Collaboration section
- `Footer` - Company information
